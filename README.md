# AI 가계부 (MVP 데모)

카드 승인 문자를 앱에 공유하면, AI가 가맹점을 분석해 자동으로 카테고리를 분류해주는 모바일 가계부 프로토타입입니다.

## 데모 링크

**https://yangtg19853585.github.io/ai-household-ledger/**

폰 브라우저(사파리)로 열어서 공유 아이콘 → "홈 화면에 추가"를 하면 앱처럼 아이콘을 눌러 실행할 수 있어요.

## 무엇을 볼 수 있나요

- **최근 30일 총 지출 / 카테고리별 막대그래프** — 식비, 카페·간식, 쇼핑, 교통, 생활, 문화·여가, 기타
- **카드 승인 문자 → 공유 → 자동 분류 흐름**
  1. "새 카드 승인 문자 받기"를 누르면 현대카드 승인 문자가 도착
  2. 문자를 "공유하기" → iOS 공유 시트에서 **AI 가계부**로 전달
  3. 처음 보는 가맹점이면 카테고리를 직접 선택 → 다음부터는 자동 분류
  4. 화면 상단에 실제 알림처럼 배너가 뜨며 분석 결과를 보여줌
- 데이터는 브라우저 `localStorage`에 저장되어, 새로고침해도 유지됩니다.

## 이게 왜 "문자 공유" 방식인가요

iOS는 다른 앱의 알림이나 문자 내용을 앱이 직접 읽는 것을 허용하지 않습니다(Android의 알림 접근 권한 같은 기능이 없음). 그래서 실제 서비스라면:

- 사용자가 카드 승인 문자를 iOS 공유 시트로 앱에 전달 (Share Extension)
- 앱이 문자 텍스트를 파싱해 가맹점·금액을 추출하고 자동/수동으로 분류
- 이후 화면은 이 데모와 동일하게 동작

이 데모는 그 흐름을 실제 문자/공유 동작 없이 화면상에서 시뮬레이션한 것입니다.

## 한계 (MVP 범위)

- 현대카드 문자 포맷만 가정한 프로토타입이며, 실제 문자/알림 연동은 되어 있지 않습니다.
- 모든 데이터는 브라우저에만 저장되며, 서버나 실제 카드 데이터를 사용하지 않습니다.
- 다른 카드사, 실제 iOS 앱(Share Extension) 구현은 다음 단계입니다.
