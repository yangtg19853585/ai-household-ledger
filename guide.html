카드 알림 한 장이
가계부 한 줄이 되기까지

카드 결제 알림을 자동으로 읽어 카테고리별로 정리해주는 가계부 앱의 프로토타입 데모입니다. 실제 카드나 알림 연동 없이, 그 흐름을 화면으로 체험해볼 수 있어요.

🔒

먼저 알아두세요

이 데모는 서버가 없어요. 지금 이 화면을 보고 있는 브라우저 안에만 데이터가 저장되고, 다른 사람이 같은 링크를 열어도 서로의 기록은 절대 보이지 않아요. 폰에서 열어본 것과 PC에서 열어본 것도 서로 다른 별개의 데이터예요.

이렇게 시작해보세요

  1. "테스트" 탭으로 이동

    하단 탭 중 🧪 테스트를 눌러요. 실제 서비스라면 자동으로 일어날 일을, 여기서는 버튼으로 직접 눌러 체험해볼 수 있어요.

  2. 카드 알림 받아보기

    "새 카드 승인 문자(또는 알림) 받기"를 누르면, 화면 위에서 알림 배너가 내려와요. AI가 가맹점을 분석하는 과정이 이어집니다.

  3. 처음 보는 가맹점이면 카테고리 선택

    이미 아는 가맹점이면 자동으로 분류되고, 처음이면 어떤 카테고리인지 직접 골라요. 편의점처럼 매번 다른 곳은 "다음부터 자동 분류" 체크를 해제해두면 매번 다시 물어봐요.

  4. "홈"에서 결과 확인

    총 지출, 예산 진행률, 카테고리별 막대그래프, 절약 인사이트, 방금 추가된 내역까지 한 번에 반영돼요.

  5. 예산을 정해보기

    홈 화면 "예산" 카드의 "설정"을 눌러 총 예산과 카테고리별 예산을 입력해보세요. 예산을 넘기는 순간 경고 알림이 자동으로 떠요.

화면 구성

🏠 홈

기간을 직접 골라 총 지출, 예산 대비 사용률, 카테고리별 막대그래프, 한 줄 절약 인사이트, 가계부 내역을 확인해요. 실제로 쌓은 데이터가 없으면 0원부터 시작합니다.

📅 기록

1월부터 이번 달까지 월별 지출 요약을 볼 수 있어요. 월을 탭하면 홈 화면이 그 기간으로 바로 이동해요. "시작일" 설정으로 매달 며칠부터 한 달로 셀지도 정할 수 있어요 (예: 25일 ~ 다음달 24일).

🧪 테스트데모 전용

실제 서비스에는 없는, 카드 알림이 오는 상황을 흉내 내는 화면이에요. 가맹점 자동분류 설정 관리와 데모 데이터 초기화도 여기 있어요.

iOS 데모 vs Android 데모

알림을 받는 방식 문자를 앱에 공유하기로 전달 (수동 1단계) 알림 접근 권한으로 자동 감지

자주 묻는 것들

초기화 버튼을 누르면 뭐가 지워지나요?

"테스트" 탭에서 추가한 최근 내역과 기간 설정만 지워져요. 가맹점 자동분류 설정, 예산 설정, "기록" 탭의 월별 더미 데이터, 월 시작일 설정은 그대로 남아요.

같은 링크를 다른 사람에게 보내면 제 데이터가 보이나요?

아니요. 서버가 없는 구조라 모든 데이터는 각자의 브라우저에만 남아요. 다만 1~6월 더미 데이터는 항상 "총 200만 원 안팎 + 비슷한 카테고리 비율"로 생성되기 때문에, 다른 사람 화면도 얼핏 비슷해 보일 수 있어요 — 실제로는 서로 완전히 독립된 데이터예요.

실제 제 카드 알림을 읽어주나요?

아니요. 이건 UX 흐름을 검증하기 위한 프로토타입이에요. 실제 카드 문자·알림 연동은 되어 있지 않고, 모든 알림은 "테스트" 탭에서 시뮬레이션한 것입니다.

알아두면 좋은 한계

  • 현대카드 알림 포맷만 가정한 프로토타입이라, 다른 카드사·실제 문자 연동은 되어 있지 않아요.
  • 모든 데이터는 브라우저에만 저장되고, 새로고침해도 유지되지만 "사이트 데이터 삭제"를 하면 사라져요.
  • 실제 네이티브 앱(iOS Share Extension / Android 알림 접근 권한) 구현은 다음 단계예요.
